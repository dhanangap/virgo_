Grid System

With Virgo, we have options to use fixed-width columns or flexible columns. The 12-column grid is suitable to build consistent layout accross devices, while with flexible-width columns is more suitable for design that's required to adjust to its content size. We can even combine both fixed and flexible columns to achieve more advanced layout.

Rows & Columns

It is important to understand the role of rows and columns in this grid system. Rows and columns are the basic building blocks of layout, with columns are placed inside a row.

A row can contain 12 fixed-width columns and (theoretically) unlimited number of flexible columns. Row is defined by using row class on an HTML element, preferably div element. 


<div class="row">
  ...we will put columns here...
</div>

By default a row fills its container width, unless specified otherwise using width utility classes.

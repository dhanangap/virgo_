Getting Started

Virgo is a modern style library to help you build websites and web applications faster, works accross multiple devices, and easier to work with teams.

Getting started with Virgo is fairly straighforward. We only need to include bundled CSS and JavaScript files in our page. In our HTML file, inside head tag, put this line to include the stylesheet: 


<link rel="stylesheet" href="/path/to/bundle.min.css">

Then just before closing body tag, put this line to include the script file: 


<script type="module" href="/path/to/bundle.min.js"></script>

That's it, and you are good to go.

Reducing File Size

You will notice that the bundled CSS file size is large. It is because the vast amount of utility classes available. While it doesn't really matter in development, it matters a lot in production environment as it will make the page loading time longer. To reduce the file size after development, please read Reducing File Size page. Typical sites built with Virgo have under 50 kilobytes CSS file size.

Working with ReactJS, Angular, and Vue

To build modern web applications, you may use frontend web application frameworks or libraries such as ReactJS, Angular, or Vue. Currently, modules for those frameworks are under development. In the meantime, you can use the style library just fine, but not with the components that require JavaScript to work.

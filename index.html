Column Width

In real-life condition, we need to adjust the column width to satisfy design requirements.
To achive this, use .col-(number) class with (number) is from 1 to 12. Total number inside a row must not exceed 12.

Column Offset

Sometimes, we need to "jump" the columns.
Offsetting allows us to start the column from specific location. Add .col-offset-(number) class to tell how far a column must jump.

Grid Spacing

While great for scaffolding, the default no-spacing columns can be improved for easier readability and clarity by adding whitespaces.
We can add .spaced class to the containing .row to give spacing between columns.
